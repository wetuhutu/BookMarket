# 首页功能测试指南

## 📋 测试概述

本文档说明如何测试首页功能，包括所需的接口、测试步骤和Mock数据。

## 🔌 所需后端接口

### 1. 获取热门分类
**接口地址**: `GET /api/books/categories`

**请求参数**: 无

**响应示例**:
```json
{
  "code": 200,
  "message": "success",
  "data": [
    {
      "id": "literature",
      "name": "文学小说",
      "icon": "📖",
      "count": 2345
    },
    {
      "id": "tech",
      "name": "计算机技术",
      "icon": "💻",
      "count": 1876
    }
  ]
}
```

---

### 2. 获取热门推荐书籍
**接口地址**: `GET /api/books/hot`

**请求参数**:
| 参数名 | 类型 | 必填 | 说明 |
|-------|------|------|------|
| limit | Number | 否 | 返回数量，默认4 |

**响应示例**:
```json
{
  "code": 200,
  "message": "success",
  "data": [
    {
      "id": 1,
      "title": "JavaScript高级程序设计",
      "author": "Nicholas C. Zakas",
      "cover": "https://example.com/book1.jpg",
      "condition": "90",
      "price": 45.00,
      "originalPrice": 99.00,
      "sellerName": "书香阁",
      "sellerLevel": "金牌卖家",
      "sellerRating": 4.9,
      "isVerified": true
    }
  ]
}
```

---

### 3. 获取书籍列表（可选）
**接口地址**: `GET /api/books`

**请求参数**:
| 参数名 | 类型 | 必填 | 说明 |
|-------|------|------|------|
| page | Number | 否 | 页码，默认1 |
| pageSize | Number | 否 | 每页数量，默认12 |
| category | String | 否 | 分类筛选 |
| sortBy | String | 否 | 排序方式（default/price_asc/price_desc/newest/sales/condition） |

**响应示例**:
```json
{
  "code": 200,
  "message": "success",
  "data": {
    "list": [],
    "total": 2456,
    "page": 1,
    "pageSize": 12
  }
}
```

## 🧪 测试步骤

### 方式一：使用Mock数据（推荐用于前端开发）

1. **启动开发服务器**
   ```bash
   npm run dev
   ```

2. **访问首页**
   - 打开浏览器访问: `http://localhost:5173/`

3. **验证功能**
   - ✅ Hero区域显示正常
   - ✅ 6个热门分类卡片显示
   - ✅ 4本热门推荐书籍显示
   - ✅ 平台特色卡片显示
   - ✅ 导航栏链接可点击
   - ✅ 按钮悬停效果正常

---

### 方式二：连接真实后端

1. **配置环境变量**
   编辑 `.env.development`:
   ```bash
   VITE_API_BASE_URL=http://your-backend-api:3000/api
   ```

2. **确保后端服务运行**
   - 后端服务已启动
   - 接口地址可访问

3. **启动前端开发服务器**
   ```bash
   npm run dev
   ```

4. **测试接口**
   - 打开浏览器开发者工具（F12）
   - 切换到 Network 标签
   - 刷新页面，查看接口请求

5. **验证数据**
   - 检查 `/api/books/categories` 返回分类数据
   - 检查 `/api/books/hot` 返回热门书籍数据

## 🐛 常见问题

### 1. 接口请求失败
**问题**: 控制台显示接口请求错误

**解决方案**:
- 检查后端服务是否启动
- 检查环境变量配置是否正确
- 检查接口地址是否可访问
- 检查跨域配置

### 2. 数据不显示
**问题**: 页面显示空白或数据未加载

**解决方案**:
- 打开浏览器开发者工具查看错误信息
- 检查API响应数据格式是否正确
- 检查前端代码是否有错误

### 3. 图片加载失败
**问题**: 书籍封面图片不显示

**解决方案**:
- 检查图片URL是否正确
- 检查图片服务器是否可访问
- 检查图片格式是否支持

### 4. 样式异常
**问题**: 页面样式显示不正确

**解决方案**:
- 检查Tailwind CSS是否正确配置
- 检查样式文件是否正确引入
- 清除浏览器缓存

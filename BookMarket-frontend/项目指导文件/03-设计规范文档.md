# 书市项目设计规范文档

## 1. 概述

本文档基于原型图提取的视觉设计规范，定义了书市项目的颜色方案、字体样式、间距规范、组件尺寸等UI设计标准，确保整个项目在视觉上保持一致性和专业性。

### 7.3 常用动画

#### 7.3.1 悬停效果
```css
/* 卡片悬停上浮 */
.card-hover {
  transition: all 0.3s ease;
}
.card-hover:hover {
  transform: translateY(-8px);
  box-shadow: 0 20px 40px rgba(0, 0, 0, 0.15);

```

#### 7.3.2 按钮悬停
```css
/* 按钮悬停上浮 */
.btn-primary {
  transition: all 0.3s ease;
}
.btn-primary:hover {
  transform: translateY(-2px);
  box-shadow: 0 10px 20px rgba(99, 102, 241, 0.3);
}
```

#### 7.3.3 导航链接下划线
```css
/* 导航链接下划线动画 */
.nav-link {
  position: relative;
}
.nav-link::after {
  content: '';
  position: absolute;
  bottom: -4px;
  left: 0;
  width: 0;
  height: 2px;
  background: #6366f1;
  transition: width 0.3s ease;
}
.nav-link:hover::after,
.nav-link.active::after {
  width: 100%;
}
```

#### 7.3.4 分类卡片悬停
```css
/* 分类卡片悬停放大 */
.category-card {
  transition: all 0.3s ease;
}
.category-card:hover {
  transform: scale(1.05);
  box-shadow: 0 10px 30px rgba(99, 102, 241, 0.2);
}
```

#### 7.3.5 缩略图悬停
```css
/* 缩略图悬停效果 */

  transition: all 0.2s ease;
}
.thumbnail:hover,
.thumbnail.active {
```

## 8. 响应式断点

### 8.1 断点定义

| 断点名称 | Tailwind类 | 最小宽度 | 应用场景 |
|---------|-----------|---------|---------|
| 移动端 | - | 0px | 手机 |
| 小屏 | sm | 640px | 大手机 |
| 中屏 | md | 768px | 平板 |
| 大屏 | lg | 1024px | 小笔记本 |
| 超大屏 | xl | 1280px | 大笔记本 |
| 2K屏 | 2xl | 1536px | 桌面显示器 |

### 8.2 响应式规范

#### 8.2.1 容器宽度
```css
/* 最大宽度容器 */
.max-w-7xl {
  max-width: 80rem; /* 1280px */
}

/* 响应式内边距 */
.px-4 {
  padding-left: 1rem;
  padding-right: 1rem;
}

@media (min-width: 640px) {
  .sm\:px-6 {
    padding-left: 1.5rem;
    padding-right: 1.5rem;
  }
}

@media (min-width: 1024px) {
  .lg\:px-8 {
    padding-left: 2rem;
    padding-right: 2rem;
  }
}
```

#### 8.2.2 网格布局
```css
/* 响应式网格 */
.grid-cols-1 {
  grid-template-columns: repeat(1, minmax(0, 1fr));
}

@media (min-width: 640px) {
  .sm\:grid-cols-2 {
    grid-template-columns: repeat(2, minmax(0, 1fr));
  }
}

@media (min-width: 1024px) {
  .lg\:grid-cols-4 {
    grid-template-columns: repeat(4, minmax(0, 1fr));
  }
}
```

#### 8.2.3 字号响应式
```css
/* 响应式字号 */
.text-4xl {
  font-size: 2.25rem;
}

@media (min-width: 768px) {
  .md\:text-5xl {
    font-size: 3rem;
  }
}

@media (min-width: 1024px) {
  .lg\:text-6xl {
    font-size: 3.75rem;
  }
```

## 10. 可访问性规范

### 10.1 颜色对比度
- 正文文字与背景对比度 ≥ 4.5:1
- 大文字（18px+）与背景对比度 ≥ 3:1
